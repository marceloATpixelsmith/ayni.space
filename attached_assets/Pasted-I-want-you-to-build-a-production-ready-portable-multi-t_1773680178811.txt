Design the architecture so we have:

1. Platform database
   Shared SaaS infrastructure tables.

2. Shipibo application database
   Contains dictionary data and logic.

3. Ayni application database
   Contains ceremony management data.

The platform database should contain tables such as:

users
organizations
organization_memberships
roles
permissions
apps
organization_app_subscriptions
subscriptions
invitations
audit_logs
feature_flags

Design the system so:

- users can belong to multiple organizations
- organizations can subscribe to multiple apps

- app access is controlled by membership + role + subscription
