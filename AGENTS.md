## CODING AGENT RULES (MANDATORY)

- Code MUST compile and be syntactically valid before committing
- ALWAYS work directly on the master branch
- NEVER create a new branch
- NEVER create a pull request
- NEVER switch branches
- ALL changes must be committed directly to master

- After every change:
  - Confirm branch is master
  - Exact files changed
  - Exact validation command run
  - Whether it passed or failed
  - Exact commit hash that was merged
- Do not introduce syntax errors
- Do not leave incomplete code
- Do not duplicate properties or variables
